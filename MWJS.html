Pros and Cons of Making Websites with Fireworks

    Pros of using fireworks to create a website
  • it is really easy to blueprint the website that we want to create
  • we can keep track of what we are doing easily with viewing the png file on chrome
  • generally, it is really easy to use it because we do not have to code anything
  • we can slice the parts that we want to separate from other things
      • Cons of using fireworks to create a website
  • there are limitations of functions
  • we can not make it as precise as other tools
